# Entrega 0 - Base de Dados

## Propósito de Avaliação

### Objetivo
Estabelecer a base de dados para análise de atrasos de voos em aeroportos dos Estados Unidos, implementando a camada **Bronze** (dados brutos) da arquitetura Medallion.

### Fonte de Dados
Os dados foram obtidos do **Bureau of Transportation Statistics (BTS)** do governo dos Estados Unidos, contendo informações históricas sobre:

- Atrasos de voos por aeroporto
- Métricas de desempenho das companhias aéreas
- Causas de atrasos (meteorologia, companhia aérea, NAS, segurança, aeronave atrasada)
- Cancelamentos e desvios de voos

### Estrutura dos Dados Brutos

O arquivo `dados_brutos.csv` contém os seguintes campos:

| Campo | Descrição |
|-------|-----------|
| year | Ano do registro |
| month | Mês do registro |
| carrier | Código da companhia aérea |
| carrier_name | Nome da companhia aérea |
| airport | Código do aeroporto |
| airport_name | Nome do aeroporto |
| arr_flights | Número de voos de chegada |
| arr_del15 | Número de voos atrasados ≥15 minutos |
| carrier_ct | Contagem de atrasos causados pela companhia |
| weather_ct | Contagem de atrasos causados por condições meteorológicas |
| nas_ct | Contagem de atrasos causados pelo Sistema de Aviação Nacional |
| security_ct | Contagem de atrasos causados por segurança |
| late_aircraft_ct | Contagem de atrasos causados por aeronave atrasada |
| arr_cancelled | Número de voos cancelados |
| arr_diverted | Número de voos desviados |
| arr_delay | Tempo total de atraso (minutos) |
| carrier_delay | Tempo de atraso causado pela companhia (minutos) |
| weather_delay | Tempo de atraso causado por meteorologia (minutos) |
| nas_delay | Tempo de atraso causado pelo NAS (minutos) |
| security_delay | Tempo de atraso causado por segurança (minutos) |
| late_aircraft_delay | Tempo de atraso causado por aeronave atrasada (minutos) |

### Armazenamento

Os dados brutos estão armazenados em:
